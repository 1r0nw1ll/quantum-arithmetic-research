## Mathematical content

### Theorem (Bateson-Cayley equivalence)

Let S_m = {(b,e) : 1 ≤ b,e ≤ m} be the QA state space and let Cay_undirected(S_m, Γ) denote the undirected graph with vertex set S_m and an edge {s, g(s)} for each s ∈ S_m and g ∈ Γ. Define the tier generator sets:

| Tier | Generator set |
|------|---------------|
| Γ_{L1}  | {T} |
| Γ_{L2a} | Γ_{L1} ∪ {scalar_k : k ∈ (ℤ/mℤ)\*, k ≠ 1} ∪ {swap} |
| Γ_{L2b} | Γ_{L2a} ∪ {scalar_k : gcd(k,m) > 1, k ≠ 1} ∪ {const_{(m,m)}} |

Then the connected components of Cay_undirected(S_m, Γ_{tier}) equal the tier-reachability classes of [191]. On m = 9 the components are:

| Tier | Component sizes | Σ|C|² (cumulative pairs) |
|------|-----------------|----------------------------|
| L1   | 24, 24, 24, 8, 1 | 1793 |
| L2a  | 72, 8, 1         | 5249 |
| L2b  | 81               | 6561 |

The non-cumulative differences are 1793 − 81 = 1712, 5249 − 1793 = 3456, 6561 − 5249 = 1312 — matching `EXPECTED_TIER_COUNTS_S9` from [191] byte-for-byte.

### Undirected convention is essential

scalar_3 and const_{(9,9)} are non-bijective on S_9 (kernels of size > 1). In the directed forward-reach convention, cosmos → satellite via scalar_3 is a one-way move. The standard Cayley-graph convention (generators closed under inverses ⇔ edges undirected) symmetrizes this, and only under that convention do the connected components match the [191] tier-reachability classes exactly.

### Proof idea (L1)

`{T}` generates the T-orbit of s_0 under successive application. Every point in Orbit(s_0) is reached from s_0 and vice-versa (orbits are cycles under the bijection T), so the component of s_0 in Cay(S_9, {T}) equals Orbit(s_0). Summing squared orbit sizes: 3·24² + 8² + 1² = 1728 + 64 + 1 = 1793.

### Proof idea (L2a)

Adding (ℤ/9ℤ)\* scalars (cyclic group of order 6, generated by 2) and the swap gives a generator set whose orbit (under composition) on any cosmos state reaches every cosmos state. On satellite states {3a, 3b : a,b ∈ {1,2,3}}, unit scalars stabilize the satellite setwise. The singularity (9,9) is fixed. Components collapse to 72 + 8 + 1 = family partition.

### Proof idea (L2b)

scalar_3 maps cosmos → satellite; const_{(9,9)} maps any state → singularity. Under undirected convention, these symmetric edges merge the three families into one component of size 81. Σ|C|² = 81² = 6561.

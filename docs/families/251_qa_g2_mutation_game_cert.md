## Claim

The integer Mutation Game framework from certs [244] (E_8) and [250]
(simply-laced ADE) extends to the first non-simply-laced type G_2 via
directed-multigraph edge counts encoding the asymmetric Cartan entries.
For Cartan `[[2, -1], [-3, 2]]`, the edge counts are `A(0 → 1) = 3`
(short-to-long) and `A(1 → 0) = 1` (long-to-short). The mutation
operators

```
s_0(p_0, p_1) = (p_1 − p_0,   p_1)
s_1(p_0, p_1) = (p_0,        3 p_0 − p_1)
```

generate, by BFS from simple roots `δ_0 = (1, 0)` and `δ_1 = (0, 1)`,
exactly 12 distinct integer populations split equally as

```
R_+ = { (1,0), (0,1), (1,1), (1,2), (1,3), (2,3) }
R_− = −R_+
```

Under the coordinate-swap bijection `(p_0, p_1) ↔ (p_1, p_0)`, the six
positive populations map to the Humphreys §12.1 simple-root positive
roots `{(1,0), (0,1), (1,1), (2,1), (3,1), (3,2)}`. Under the Humphreys
Gram matrix `G_sr = [[2, -3], [-3, 6]]`, the image norms split as three
short (norm 2) and three long (norm 6) — matching Humphreys Table 1 for
G_2 exactly. The Weyl group generated by `{s_0, s_1}` is dihedral of
order 12: `s_0^2 = s_1^2 = I` and `(s_0 s_1)^6 = I` strictly (fails for
every `k ∈ {1, …, 5}`).

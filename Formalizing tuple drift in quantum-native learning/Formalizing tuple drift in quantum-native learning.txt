where:
    • $\hat{p}_{\text{legal}} \in [0,1]$ (predicted legality probability) 
    • $\hat{f}_{\text{fail}}$ (predicted fail_type distribution) 
    • $\hat{s}' \in \mathcal{S}_q$ (predicted next state if legal) 
    • $\hat{V}_k(s; \tau) \in [0,1]$ (predicted probability of return-in-k to $\mathcal{R}^*$) 
Training Objective (Supervised): Minimize prediction error on legality and fail_type: $$\mathcal{L}{\text{legality}}(\psi) = \sum{(s,g,\text{outcome}) \in \mathcal{D}} \ell(\hat{p}_{\text{legal}}, \mathbb{1}[\text{outcome} = \text{legal}])$$
Training Objective (Value Learning): Use TD-learning or fitted value iteration: $$\mathcal{L}{\text{value}}(\psi) = \sum{i} \big(\hat{V}_k(s_i; \tau) - y_i\big)^2$$ where $y_i$ is the empirical return-in-k outcome.

2.4 Policy: Meta-Learning Over Generators
Definition (Generator Selection Policy): A policy $\pi_\theta(g \mid s, \tau)$ over generators given:
    • Current state $s$ 
    • Task specification $\tau = (\mathcal{R}^*, k, \Sigma_{\text{allowed}})$ 
Training Data: Episodes ${(s_0, g_1, s_1, \dots, s_T, r_T)}$ where:
    • $r_T = 1$ if return-in-k succeeded 
    • $r_T = 0$ otherwise 
Training Objective (Policy Gradient / RL): Maximize expected return-in-k success: $$J(\theta) = \mathbb{E}{\tau \sim \mathcal{T}, , \text{episode} \sim \pi\theta} [r_T]$$
Update via REINFORCE, PPO, or other policy gradient method.

2.5 Planning: Inference Using Learned Structure
Definition (QA Planner): Given world model $\hat{M}\psi$, policy $\pi\theta$, and task $\tau$:
$$\texttt{plan}(s_0, \tau; \hat{M}\psi, \pi\theta) \to (g_1, \dots, g_T)$$
Planner Options:
    1. Policy rollout: sample $g_t \sim \pi_\theta(\cdot \mid s_t, \tau)$ 
    2. Model-predictive search: use $\hat{M}_\psi$ for lookahead (A*, MCTS) 
    3. Hybrid: use $\pi_\theta$ to guide search with $\hat{M}_\psi$ for pruning 
Critical Distinction: Planning is inference, not learning. It uses $\hat{M}\psi$ and $\pi\theta$ without updating them.

2.6 Generalization: The Learning Test
Definition (Generalization): The world model $\hat{M}_\psi$ generalizes if:
$$\text{Prediction accuracy on } (s, g) \notin \mathcal{D} \text{ exceeds baseline}$$
Baselines:
    • Random guessing 
    • Nearest-neighbor from $\mathcal{D}$ 
    • No-op (assume all illegal) 
What Constitutes Strong Generalization:
    • Correctly predict legality for unseen states in novel components 
    • Correctly predict return-in-k for unseen targets 
    • Correctly infer fail_type from invariant structure 

2.7 The Learning Loop (Formal)
Initialize: M̂_ψ (world model), π_θ (policy)

for episode = 1, 2, ... :
    Sample task τ = (R*, k, Σ)
    Initialize s₀ ~ S_q
    
    for t = 0, 1, ..., T-1:
        # Policy selects generator
        g_t ~ π_θ(· | s_t, τ)
        
        # Execute in environment
        outcome_t, s_{t+1} = execute(s_t, g_t)
        
        # Log experience
        D ← D ∪ {(s_t, g_t, outcome_t, fail_type, ΔI)}
        
        # Check termination
        if s_{t+1} ∈ R* or t >= k:
            break
    
    # Compute return
    r_T = 1[s_T ∈ R* and T ≤ k]
    
    # Update world model (supervised)
    ψ ← ψ - ∇_ψ L_legality(ψ; D) - ∇_ψ L_value(ψ; D)
    
    # Update policy (RL)
    θ ← θ + ∇_θ J(θ)

class QAWorldModel(nn.Module):
    """Learned predictor of legality, transitions, returnability"""
    
    def __init__(self, state_dim: int, gen_dim: int, hidden_dim: int):
        super().__init__()
        self.encoder = nn.Sequential(
            nn.Linear(state_dim + gen_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU()
        )
        
        # Legality head
        self.legal_head = nn.Linear(hidden_dim, 1)
        
        # Fail type head
        self.fail_type_head = nn.Linear(hidden_dim, 5)  # |fail_types|
        
        # Next state head
        self.next_state_head = nn.Linear(hidden_dim, state_dim)
        
        # Value head (return-in-k predictor)
        self.value_head = nn.Linear(hidden_dim, 1)
    
    def forward(self, state_embed, gen_embed):
        """
        Predict: (legal_prob, fail_type_dist, next_state, V_k)
        """
        x = torch.cat([state_embed, gen_embed], dim=-1)
        h = self.encoder(x)
        
        p_legal = torch.sigmoid(self.legal_head(h))
        fail_dist = torch.softmax(self.fail_type_head(h), dim=-1)
        next_state = self.next_state_head(h)
        V_k = torch.sigmoid(self.value_head(h))
        
        return p_legal, fail_dist, next_state, V_k
    
    def train_step(self, batch, optimizer):
        """
        Supervised learning on legality, fail_type, value
        """
        states, gens, legal_labels, fail_labels, values = batch
        
        p_legal, fail_dist, _, V_k = self(states, gens)
        
        # Legality loss (BCE)
        loss_legal = F.binary_cross_entropy(p_legal, legal_labels)
        
        # Fail type loss (CE, only on illegal moves)
        loss_fail = F.cross_entropy(fail_dist[~legal_labels], 
                                    fail_labels[~legal_labels])
        
        # Value loss (MSE)
        loss_value = F.mse_loss(V_k, values)
        
        loss = loss_legal + loss_fail + loss_value
        
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
        
        return loss.item()

class RMLPolicy(nn.Module):
    """Learned policy over generator alphabet"""
    
    def __init__(self, state_dim: int, num_generators: int, hidden_dim: int):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(state_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, num_generators)
        )
    
    def forward(self, state_embed):
        """Return logits over generator alphabet"""
        return self.net(state_embed)
    
    def select_generator(self, state_embed) -> int:
        """Sample generator index from policy"""
        logits = self(state_embed)
        dist = torch.distributions.Categorical(logits=logits)
        return dist.sample()
    
    def update_from_episode(self, trajectory, reward, optimizer):
        """Policy gradient update (REINFORCE)"""
        states, actions = zip(*trajectory)
        
        log_probs = []
        for state, action in zip(states, actions):
            logits = self(state)
            dist = torch.distributions.Categorical(logits=logits)
            log_probs.append(dist.log_prob(action))
        
        # REINFORCE: ∇J = E[∑ log π(a|s) * R]
        loss = -sum(log_probs) * reward
        
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()

class QABenchmark:
    """Generate reproducible benchmark statistics for QA caps"""

    def enumerate_cap(self, N: int) -> List[QAState]:
        """
        Enumerate all QA states in Caps(N,N).
        Returns: list of valid QA states satisfying b²+e² = d²+a² = N
        """
        states = []
        
        # Brute force search (optimize later if needed)
        max_val = int(N**0.5) + 1
        
        for b in range(1, max_val):
            for e in range(1, max_val):
                if b**2 + e**2 == N:
                    for d in range(1, max_val):
                        for a in range(1, max_val):
                            if d**2 + a**2 == N:
                                state = self.oracle._construct_state(b, e, d, a)
                                states.append(state)
        
        return states
    
    def compute_topology_stats(self, states: List[QAState], 
                              generators: List[str]) -> dict:
        """
        Compute:
        - #states
        - #edges (legal transitions)
        - #failures (by type)
        - #SCCs (via Tarjan)
        - max SCC size
        """
        num_states = len(states)
        
        # Count edges and failures
        legal_edges = 0
        fail_counts = defaultdict(int)
        
        for s in states:
            for g in generators:
                if self.oracle.is_legal(s, g):
                    legal_edges += 1
                else:
                    fail_type = self.oracle.get_fail_type(s, g)
                    fail_counts[fail_type.value] += 1
        
        # Compute SCCs (Tarjan's algorithm)
        sccs = self._compute_sccs(states, generators)
        num_sccs = len(sccs)
        max_scc_size = max(len(scc) for scc in sccs) if sccs else 0
        
        return {
            'num_states': num_states,
            'num_edges': legal_edges,
            'num_failures': sum(fail_counts.values()),
            'fail_type_counts': dict(fail_counts),
            'num_sccs': num_sccs,
            'max_scc_size': max_scc_size
        }
    
    def compute_return_in_k_stats(self, states: List[QAState], 
                                  target_class: Set[QAState],
                                  k: int, generators: List[str]) -> dict:
        """
        Compute return-in-k success rate across all states.
        """
        successes = sum(
            1 for s in states 
            if self.oracle.return_in_k(s, target_class, k, generators)
        )
        
        return {
            'success_rate': successes / len(states),
            'num_successes': successes,
            'num_states': len(states),
            'k': k
        }
    
    def _compute_sccs(self, states: List[QAState], 
                     generators: List[str]) -> List[Set[QAState]]:
        """Tarjan's algorithm for SCC computation"""
        # Build adjacency list
        graph = defaultdict(list)
        for s in states:
            for g in generators:
                if self.oracle.is_legal(s, g):
                    next_s = self.oracle.step(s, g)
                    graph[s].append(next_s)
        
        # Tarjan's SCC algorithm
        index_counter = [0]
        stack = []
        lowlinks = {}
        index = {}
        on_stack = defaultdict(bool)
        sccs = []
        
        def strongconnect(node):
            index[node] = index_counter[0]
            lowlinks[node] = index_counter[0]
            index_counter[0] += 1
            on_stack[node] = True
            stack.append(node)
            
            for neighbor in graph[node]:
                if neighbor not in index:
                    strongconnect(neighbor)
                    lowlinks[node] = min(lowlinks[node], lowlinks[neighbor])
                elif on_stack[neighbor]:
                    lowlinks[node] = min(lowlinks[node], index[neighbor])
            
            if lowlinks[node] == index[node]:
                scc = set()
                while True:
                    w = stack.pop()
                    on_stack[w] = False
                    scc.add(w)
                    if w == node:
                        break
                sccs.append(scc)
        
        for node in states:
            if node not in index:
                strongconnect(node)
        
        return sccs
    
    def generate_paper1_table(self, caps: List[int], 
                            generator_sets: List[List[str]]) -> str:
        """
        Generate LaTeX table for Paper 1.
        """
        rows = []
        
        for N in caps:
            states = self.enumerate_cap(N)
            
            for gen_set in generator_sets:
                stats = self.compute_topology_stats(states, gen_set)
                
                row = f"Caps({N},{N}) & {len(gen_set)} & {stats['num_states']} & " \
                      f"{stats['num_edges']} & {stats['num_failures']} & " \
                      f"{stats['num_sccs']} & {stats['max_scc_size']} \\\\"
                rows.append(row)
        
        table = "\\begin{tabular}{lcccccc}\n"
        table += "Cap & $|\\Sigma|$ & \\#States & \\#Edges & \\#Fail & \\#SCC & Max-SCC \\\\\n"
        table += "\\hline\n"
        table += "\n".join(rows)
        table += "\n\\end{tabular}"
        
        return table

Layer 1 Oracle: Canonical QA Transition System
Implements exact semantics from Caps phase-transition experiments.

State: primitive (b,e), derive 21-element invariant packet
Moves: σ, μ, λₖ, ν as specified
Caps: {(b,e) : 1 ≤ b ≤ N, 1 ≤ e ≤ N}

from dataclasses import dataclass
from enum import Enum
from typing import Optional, Set, List, Tuple, Dict

class FailType(Enum):
    """Deterministic failure classifications"""
    OUT_OF_BOUNDS = "OUT_OF_BOUNDS"
    PARITY = "PARITY"
    PHASE_VIOLATION = "PHASE_VIOLATION"
    INVARIANT = "INVARIANT"
    REDUCTION = "REDUCTION"

@dataclass(frozen=True)
class QAState:
    """
    Canonical QA state: primitive (b,e) + derived 21-element packet.
    Immutable for hashing and set membership.
    """
    # Primitives
    b: int
    e: int
    
    # Derived (d, a)
    d: int  # = b + e
    a: int  # = e + d = b + 2e
    
    # Derived invariants (21-element packet)
    B: int  # = b²
    E: int  # = e²
    D: int  # = d²
    A: int  # = a²
    X: int  # = e*d
    C: int  # = 2*e*d
    F: int  # = b*a
    G: int  # = D + E = d² + e²
    L: int  # = C*F/12
    H: int  # = C + F
    I: int  # = C - F (must be positive)
    J: int  # = d*b
    K: int  # = d*a
    W: int  # = d(e+a) = K + C/2
    Y: int  # = A - D
    Z: int  # = E + K = e² + (a*d)
    h: float  # = d * sqrt(a*b) (semi-minor diameter)
    
    # Phase annotations (not constraints for q_def="none")
    phi_9: int   # = digital_root(a)
    phi_24: int  # = a mod 24
    
    def __hash__(self):
        return hash((self.b, self.e))
    
    def __eq__(self, other):
        return self.b == other.b and self.e == other.e
    
    @property
    def primitive(self) -> Tuple[int, int]:
        return (self.b, self.e)

@dataclass
class Failure:
    """Typed failure result with invariant diagnostics"""
    state: QAState
    move: str
    fail_type: FailType
    invariant_delta: Dict[str, any]
    message: str

def digital_root(n: int) -> int:
    """
    Compute digital root (iterative sum of digits until single digit).
    Equivalent to (n-1) mod 9 + 1 for n > 0.
    """
    if n == 0:
        return 0
    return (n - 1) % 9 + 1

def construct_qa_state(b: int, e: int) -> QAState:
    """
    Build complete QA state from primitive (b,e).
    Implements exact 21-element formula packet.
    """
    # Derived base
    d = b + e
    a = e + d  # = b + 2e
    
    # Squares
    B = b * b
    E = e * e
    D = d * d
    A = a * a
    
    # Products and combinations
    X = e * d
    C = 2 * e * d
    F = b * a
    G = D + E
    L = (C * F) // 12  # Integer division
    H = C + F
    I = C - F
    J = d * b
    K = d * a
    W = d * (e + a)  # = K + X
    Y = A - D
    Z = E + K
    h = d * sqrt(a * b)  # Semi-minor diameter (float)
    
    # Phase annotations
    phi_9 = digital_root(a)
    phi_24 = a % 24
    
    return QAState(
        b=b, e=e, d=d, a=a,
        B=B, E=E, D=D, A=A,
        X=X, C=C, F=F, G=G,
        L=L, H=H, I=I, J=J,
        K=K, W=W, Y=Y, Z=Z, h=h,
        phi_9=phi_9, phi_24=phi_24
    )

class QAOracle:
    """
    Canonical QA transition oracle.
    Implements exact Caps experiment semantics.
    """
    
    def __init__(self, N: int, q_def: str = "none"):
        """
        Args:
            N: Caps bound (1 ≤ b,e ≤ N)
            q_def: Phase constraint ("none", "phi_24", "phi_9", "both")
        """
        self.N = N
        self.q_def = q_def
        
        # Caches for performance
        self._cache_legal: Dict[Tuple[QAState, str], bool] = {}
        self._cache_step: Dict[Tuple[QAState, str], QAState | Failure] = {}
    
    # ------------------------------------------------------------------------
    # Core Predicates
    # ------------------------------------------------------------------------
    
    def is_legal(self, state: QAState, move: str, k: int = 2) -> bool:
        """
        Ground truth legality check.
        
        Args:
            state: Current QA state
            move: One of {"sigma", "mu", "lambda", "nu"}
            k: Scale factor for lambda (default 2)
        """
        cache_key = (state, f"{move}_{k}" if move == "lambda" else move)
        if cache_key in self._cache_legal:
            return self._cache_legal[cache_key]
        
        result = self.step(state, move, k)
        is_legal = not isinstance(result, Failure)
        
        self._cache_legal[cache_key] = is_legal
        return is_legal
    
    def step(self, state: QAState, move: str, k: int = 2) -> QAState | Failure:
        """
        Execute move on state.
        
        Args:
            state: Current state
            move: One of {"sigma", "mu", "lambda", "nu"}
            k: Scale factor for lambda
            
        Returns:
            Next QAState if legal, Failure if illegal
        """
        cache_key = (state, f"{move}_{k}" if move == "lambda" else move)
        if cache_key in self._cache_step:
            return self._cache_step[cache_key]
        
        if move == "sigma":
            result = self._apply_sigma(state)
        elif move == "mu":
            result = self._apply_mu(state)
        elif move == "lambda":
            result = self._apply_lambda(state, k)
        elif move == "nu":
            result = self._apply_nu(state)
        else:
            raise ValueError(f"Unknown move: {move}")
        
        self._cache_step[cache_key] = result
        return result
    
    def get_fail_type(self, state: QAState, move: str, k: int = 2) -> Optional[FailType]:
        """Return fail_type if illegal, None if legal"""
        result = self.step(state, move, k)
        if isinstance(result, Failure):
            return result.fail_type
        return None
    
    # ------------------------------------------------------------------------
    # Move Implementations (Exact Caps Semantics)
    # ------------------------------------------------------------------------
    
    def _apply_sigma(self, s: QAState) -> QAState | Failure:
        """σ: (b,e) → (b, e+1)"""
        b, e = s.b, s.e
        e_next = e + 1
        
        # Bounds check
        if e_next > self.N:
            return Failure(
                state=s,
                move="sigma",
                fail_type=FailType.OUT_OF_BOUNDS,
                invariant_delta={'e': e_next - e},
                message=f"e+1 = {e_next} exceeds N = {self.N}"
            )
        
        # Construct next state
        next_s = construct_qa_state(b, e_next)
        
        # Phase check (if constrained)
        if not self._check_phase_constraint(s, next_s):
            return Failure(
                state=s,
                move="sigma",
                fail_type=FailType.PHASE_VIOLATION,
                invariant_delta=self._phase_delta(s, next_s),
                message="Phase constraint violated"
            )
        
        return next_s
    
    def _apply_mu(self, s: QAState) -> QAState | Failure:
        """μ: (b,e) → (e,b)"""
        b, e = s.b, s.e
        
        # Bounds check (automatic for square Caps)
        if e > self.N or b > self.N:
            return Failure(
                state=s,
                move="mu",
                fail_type=FailType.OUT_OF_BOUNDS,
                invariant_delta={},
                message="Swap exceeds bounds"
            )
        
        next_s = construct_qa_state(e, b)
        
        if not self._check_phase_constraint(s, next_s):
            return Failure(
                state=s,
                move="mu",
                fail_type=FailType.PHASE_VIOLATION,
                invariant_delta=self._phase_delta(s, next_s),
                message="Phase constraint violated"
            )
        
        return next_s
    
    def _apply_lambda(self, s: QAState, k: int) -> QAState | Failure:
        """λₖ: (b,e) → (kb, ke)"""
        b, e = s.b, s.e
        b_next = k * b
        e_next = k * e
        
        # Bounds check
        if b_next > self.N or e_next > self.N:
            return Failure(
                state=s,
                move=f"lambda_{k}",
                fail_type=FailType.OUT_OF_BOUNDS,
                invariant_delta={'b': b_next - b, 'e': e_next - e},
                message=f"Scaled state ({b_next},{e_next}) exceeds N = {self.N}"
            )
        
        next_s = construct_qa_state(b_next, e_next)
        
        if not self._check_phase_constraint(s, next_s):
            return Failure(
                state=s,
                move=f"lambda_{k}",
                fail_type=FailType.PHASE_VIOLATION,
                invariant_delta=self._phase_delta(s, next_s),
                message="Phase constraint violated"
            )
        
        return next_s
    
    def _apply_nu(self, s: QAState) -> QAState | Failure:
        """ν: (b,e) → (b/2, e/2) if both even"""
        b, e = s.b, s.e
        
        # Parity check
        if b % 2 != 0 or e % 2 != 0:
            return Failure(
                state=s,
                move="nu",
                fail_type=FailType.PARITY,
                invariant_delta={'b_parity': b % 2, 'e_parity': e % 2},
                message="ν requires both b,e even"
            )
        
        b_next = b // 2
        e_next = e // 2
        
        # Positivity check (should always pass given b,e > 0)
        if b_next < 1 or e_next < 1:
            return Failure(
                state=s,
                move="nu",
                fail_type=FailType.OUT_OF_BOUNDS,
                invariant_delta={},
                message="Contraction would violate positivity"
            )
        
        next_s = construct_qa_state(b_next, e_next)
        
        if not self._check_phase_constraint(s, next_s):
            return Failure(
                state=s,
                move="nu",
                fail_type=FailType.PHASE_VIOLATION,
                invariant_delta=self._phase_delta(s, next_s),
                message="Phase constraint violated"
            )
        
        return next_s
    
    # ------------------------------------------------------------------------
    # Phase Constraint Checking
    # ------------------------------------------------------------------------
    
    def _check_phase_constraint(self, s1: QAState, s2: QAState) -> bool:
        """Check if transition preserves phase constraint"""
        if self.q_def == "none":
            return True  # Unconstrained
        elif self.q_def == "phi_24":
            return s1.phi_24 == s2.phi_24
        elif self.q_def == "phi_9":
            return s1.phi_9 == s2.phi_9
        elif self.q_def == "both":
            return s1.phi_9 == s2.phi_9 and s1.phi_24 == s2.phi_24
        else:
            raise ValueError(f"Unknown q_def: {self.q_def}")
    
    def _phase_delta(self, s1: QAState, s2: QAState) -> Dict[str, int]:
        """Compute phase difference"""
        return {
            'phi_9': s2.phi_9 - s1.phi_9,
            'phi_24': s2.phi_24 - s1.phi_24
        }
    
    # ------------------------------------------------------------------------
    # Reachability (Return-in-k)
    # ------------------------------------------------------------------------
    
    def return_in_k(self, state: QAState, target_class: Set[QAState],
                    k: int, generators: List[Tuple[str, int]]) -> bool:
        """
        Exact reachability check via bounded BFS.
        
        Args:
            state: Starting state
            target_class: Set of goal states
            k: Maximum depth
            generators: List of (move_name, k_param) tuples
                       e.g., [("sigma", 2), ("mu", 2), ("lambda", 2), ("nu", 2)]
        
        Returns:
            True iff ∃ path of length ≤k to target_class
        """
        if state in target_class:
            return True
        
        visited = {state}
        frontier = [(state, 0)]
        
        while frontier:
            s, depth = frontier.pop(0)
            
            if depth >= k:
                continue
            
            for move, k_param in generators:
                if self.is_legal(s, move, k_param):
                    next_s = self.step(s, move, k_param)
                    
                    if next_s in target_class:
                        return True
                    
                    if next_s not in visited:
                        visited.add(next_s)
                        frontier.append((next_s, depth + 1))
        
        return False
    
    # ------------------------------------------------------------------------
    # Topology Analysis
    # ------------------------------------------------------------------------
    
    def compute_scc_id(self, state: QAState, generators: List[Tuple[str, int]],
                      max_depth: int = 100) -> str:
        """
        Approximate SCC ID via reachable set hash.
        For exact SCCs, use Tarjan on full graph.
        """
        reachable = self._compute_reachable_set(state, generators, max_depth)
        canonical = min(reachable, key=lambda s: s.primitive)
        return hashlib.md5(str(canonical.primitive).encode()).hexdigest()[:8]
    
    def _compute_reachable_set(self, state: QAState, 
                              generators: List[Tuple[str, int]],
                              max_depth: int) -> Set[QAState]:
        """BFS reachable set computation"""
        visited = {state}
        frontier = [state]
        depth = 0
        
        while frontier and depth < max_depth:
            next_frontier = []
            for s in frontier:
                for move, k_param in generators:
                    if self.is_legal(s, move, k_param):
                        next_s = self.step(s, move, k_param)
                        if next_s not in visited:
                            visited.add(next_s)
                            next_frontier.append(next_s)
            frontier = next_frontier
            depth += 1
        
        return visited

class QABenchmark:
    """Generate reproducible benchmark statistics for Caps(N,N)"""
    
    def __init__(self, oracle: QAOracle):
        self.oracle = oracle
    
    def enumerate_caps(self) -> List[QAState]:
        """
        Enumerate all states in Caps(N,N).
        Returns: [(b,e) : 1 ≤ b ≤ N, 1 ≤ e ≤ N]
        """
        N = self.oracle.N
        states = []
        
        for b in range(1, N + 1):
            for e in range(1, N + 1):
                state = construct_qa_state(b, e)
                states.append(state)
        
        return states
    
    def compute_topology_stats(self, states: List[QAState],
                              generators: List[Tuple[str, int]]) -> Dict:
        """
        Compute topology statistics:
        - #states, #edges, #failures (by type)
        - #SCCs, max SCC size
        
        Args:
            generators: [(move, k_param), ...] e.g., [("sigma", 2), ("lambda", 2)]
        """
        num_states = len(states)
        
        # Count edges and failures
        legal_edges = 0
        fail_counts = defaultdict(int)
        
        for s in states:
            for move, k_param in generators:
                if self.oracle.is_legal(s, move, k_param):
                    legal_edges += 1
                else:
                    fail_type = self.oracle.get_fail_type(s, move, k_param)
                    if fail_type:
                        fail_counts[fail_type.value] += 1
        
        # Compute SCCs via Tarjan
        sccs = self._compute_sccs(states, generators)
        num_sccs = len(sccs)
        max_scc_size = max(len(scc) for scc in sccs) if sccs else 0
        
        return {
            'num_states': num_states,
            'num_edges': legal_edges,
            'num_failures': sum(fail_counts.values()),
            'fail_type_counts': dict(fail_counts),
            'num_sccs': num_sccs,
            'max_scc_size': max_scc_size,
            'scc_histogram': self._scc_histogram(sccs)
        }
    
    def _compute_sccs(self, states: List[QAState],
                     generators: List[Tuple[str, int]]) -> List[Set[QAState]]:
        """
        Tarjan's algorithm for exact SCC computation.
        """
        # Build adjacency list
        graph = defaultdict(list)
        state_set = set(states)
        
        for s in states:
            for move, k_param in generators:
                if self.oracle.is_legal(s, move, k_param):
                    next_s = self.oracle.step(s, move, k_param)
                    if next_s in state_set:  # Stay within enumerated set
                        graph[s].append(next_s)
        
        # Tarjan's algorithm
        index_counter = [0]
        stack = []
        lowlinks = {}
        index = {}
        on_stack = defaultdict(bool)
        sccs = []
        
        def strongconnect(node):
            index[node] = index_counter[0]
            lowlinks[node] = index_counter[0]
            index_counter[0] += 1
            on_stack[node] = True
            stack.append(node)
            
            for neighbor in graph[node]:
                if neighbor not in index:
                    strongconnect(neighbor)
                    lowlinks[node] = min(lowlinks[node], lowlinks[neighbor])
                elif on_stack[neighbor]:
                    lowlinks[node] = min(lowlinks[node], index[neighbor])
            
            if lowlinks[node] == index[node]:
                scc = set()
                while True:
                    w = stack.pop()
                    on_stack[w] = False
                    scc.add(w)
                    if w == node:
                        break
                sccs.append(scc)
        
        for node in states:
            if node not in index:
                strongconnect(node)
        
        return sccs
    
    def _scc_histogram(self, sccs: List[Set[QAState]]) -> Dict[int, int]:
        """Return histogram of SCC sizes"""
        hist = defaultdict(int)
        for scc in sccs:
            hist[len(scc)] += 1
        return dict(hist)
    
    def generate_paper1_table(self, caps_list: List[int],

            ]

                row = (f"Caps({N},{N}) & {config['name']} & {stats['num_states']} & "
                       f"{stats['num_edges']} & {stats['num_failures']} & "
                       f"{stats['num_sccs']} & {stats['max_scc_size']} \\\\")
                rows.append(row)
        
        table = "\\begin{tabular}{lcccccc}\n"
        table += "Cap & $\\Sigma$ & \\#States & \\#Edges & \\#Fail & \\#SCC & Max-SCC \\\\\n"

def digital_root(n: int) -> int:
    """
    Compute digital root (iterative sum of digits until single digit).
    Equivalent to (n-1) mod 9 + 1 for n > 0.
    """
    if n == 0:
        return 0
    return (n - 1) % 9 + 1

class QAOracle:
    """
    Canonical QA transition oracle.
    Implements exact Caps experiment semantics.
    """
    
    def __init__(self, N: int, q_def: str = "none"):
        """
        Args:
            N: Caps bound (1 ≤ b,e ≤ N)
            q_def: Phase constraint ("none", "phi_24", "phi_9", "both")
        """
        self.N = N
        self.q_def = q_def
        
        # Caches for performance
        self._cache_legal: Dict[Tuple[QAState, str, int], bool] = {}
        self._cache_step: Dict[Tuple[QAState, str, int], QAState | Failure] = {}
    
    # ------------------------------------------------------------------------
    # Core Predicates
    # ------------------------------------------------------------------------
    
    def is_legal(self, state: QAState, move: str, k: int = 2) -> bool:
        """
        Ground truth legality check.
        
        Args:
            state: Current QA state
            move: One of {"sigma", "mu", "lambda2", "nu"}
            k: Scale factor for lambda2 (always 2 for canonical experiments)
        """
        cache_key = (state, move, k)
        if cache_key in self._cache_legal:
            return self._cache_legal[cache_key]
        
        result = self.step(state, move, k)
        is_legal = not isinstance(result, Failure)
        
        self._cache_legal[cache_key] = is_legal
        return is_legal
    
    def step(self, state: QAState, move: str, k: int = 2) -> QAState | Failure:
        """
        Execute move on state.
        
        Args:
            state: Current state
            move: One of {"sigma", "mu", "lambda2", "nu"}
            k: Scale factor for lambda2
            
        Returns:
            Next QAState if legal, Failure if illegal
        """
        cache_key = (state, move, k)
        if cache_key in self._cache_step:
            return self._cache_step[cache_key]
        
        if move == "sigma":
            result = self._apply_sigma(state)
        elif move == "mu":
            result = self._apply_mu(state)
        elif move == "lambda2":
            result = self._apply_lambda2(state, k)
        elif move == "nu":
            result = self._apply_nu(state)
        else:
            raise ValueError(f"Unknown move: {move}")
        
        self._cache_step[cache_key] = result
        return result
    
    def get_fail_type(self, state: QAState, move: str, k: int = 2) -> Optional[FailType]:
        """Return fail_type if illegal, None if legal"""
        result = self.step(state, move, k)
        if isinstance(result, Failure):
            return result.fail_type
        return None

    # Move Implementations (Exact Caps Semantics)
    # ------------------------------------------------------------------------
    
    def _apply_sigma(self, s: QAState) -> QAState | Failure:
        """σ: (b,e) → (b, e+1)"""
        b, e = s.b, s.e
        e_next = e + 1
        
        # Bounds check
        if e_next > self.N:
            return Failure(
                state=s,
                move="sigma",
                fail_type=FailType.OUT_OF_BOUNDS,
                invariant_delta={'e': e_next - e},
                message=f"e+1 = {e_next} exceeds N = {self.N}"
            )
        
        # Construct next state
        next_s = construct_qa_state(b, e_next)
        
        # Phase check (if constrained)
        if not self._check_phase_constraint(s, next_s):
            return Failure(
                state=s,
                move="sigma",
                fail_type=FailType.PHASE_VIOLATION,
                invariant_delta=self._phase_delta(s, next_s),
                message="Phase constraint violated"
            )
        
        return next_s
    
    def _apply_mu(self, s: QAState) -> QAState | Failure:
        """μ: (b,e) → (e,b)"""
        b, e = s.b, s.e
        
        # Bounds check (automatic for square Caps)
        if e > self.N or b > self.N:
            return Failure(
                state=s,
                move="mu",
                fail_type=FailType.OUT_OF_BOUNDS,
                invariant_delta={},
                message="Swap exceeds bounds"
            )
        
        next_s = construct_qa_state(e, b)
        
        if not self._check_phase_constraint(s, next_s):
            return Failure(
                state=s,
                move="mu",
                fail_type=FailType.PHASE_VIOLATION,
                invariant_delta=self._phase_delta(s, next_s),
                message="Phase constraint violated"
            )
        
        return next_s
    
    def _apply_lambda2(self, s: QAState, k: int = 2) -> QAState | Failure:
        """λ₂: (b,e) → (2b, 2e)"""
        b, e = s.b, s.e
        b_next = k * b
        e_next = k * e
        
        # Bounds check
        if b_next > self.N or e_next > self.N:
            return Failure(
                state=s,
                move="lambda2",
                fail_type=FailType.OUT_OF_BOUNDS,
                invariant_delta={'b': b_next - b, 'e': e_next - e},
                message=f"Scaled state ({b_next},{e_next}) exceeds N = {self.N}"
            )
        
        next_s = construct_qa_state(b_next, e_next)
        
        if not self._check_phase_constraint(s, next_s):
            return Failure(
                state=s,
                move="lambda2",
                fail_type=FailType.PHASE_VIOLATION,
                invariant_delta=self._phase_delta(s, next_s),
                message="Phase constraint violated"
            )
        
        return next_s
    
    def _apply_nu(self, s: QAState) -> QAState | Failure:
        """ν: (b,e) → (b/2, e/2) if both even"""
        b, e = s.b, s.e
        
        # Parity check
        if b % 2 != 0 or e % 2 != 0:
            return Failure(
                state=s,
                move="nu",
                fail_type=FailType.PARITY,
                invariant_delta={'b_parity': b % 2, 'e_parity': e % 2},
                message="ν requires both b,e even"
            )
        
        b_next = b // 2
        e_next = e // 2
        
        # Positivity check (should always pass given b,e > 0)
        if b_next < 1 or e_next < 1:
            return Failure(
                state=s,
                move="nu",
                fail_type=FailType.OUT_OF_BOUNDS,
                invariant_delta={},
                message="Contraction would violate positivity"
            )
        
        next_s = construct_qa_state(b_next, e_next)
        
        if not self._check_phase_constraint(s, next_s):
            return Failure(
                state=s,
                move="nu",
                fail_type=FailType.PHASE_VIOLATION,
                invariant_delta=self._phase_delta(s, next_s),
                message="Phase constraint violated"
            )
        
        return next_s
    
    # ------------------------------------------------------------------------
    # Phase Constraint Checking
    # ------------------------------------------------------------------------
    
    def _check_phase_constraint(self, s1: QAState, s2: QAState) -> bool:
        """Check if transition preserves phase constraint"""
        if self.q_def == "none":
            return True  # Unconstrained
        elif self.q_def == "phi_24":
            return s1.phi_24 == s2.phi_24
        elif self.q_def == "phi_9":
            return s1.phi_9 == s2.phi_9
        elif self.q_def == "both":
            return s1.phi_9 == s2.phi_9 and s1.phi_24 == s2.phi_24
        else:
            raise ValueError(f"Unknown q_def: {self.q_def}")
    
    def _phase_delta(self, s1: QAState, s2: QAState) -> Dict[str, int]:
        """Compute phase difference"""
        return {
            'phi_9': s2.phi_9 - s1.phi_9,
            'phi_24': s2.phi_24 - s1.phi_24
        }
    
    # ------------------------------------------------------------------------
    # Reachability (Return-in-k)
    # ------------------------------------------------------------------------
    
    def return_in_k(self, state: QAState, target_class: Set[QAState],
                    k: int, generators: List[Tuple[str, int]]) -> bool:
        """
        Exact reachability check via bounded BFS.
        
        Args:
            state: Starting state
            target_class: Set of goal states
            k: Maximum depth
            generators: List of (move_name, k_param) tuples
                       e.g., [("sigma", 2), ("mu", 2), ("lambda2", 2), ("nu", 2)]
        
        Returns:
            True iff ∃ path of length ≤k to target_class
        """
        if state in target_class:
            return True
        
        visited = {state}
        frontier = [(state, 0)]
        
        while frontier:
            s, depth = frontier.pop(0)
            
            if depth >= k:
                continue
            
            for move, k_param in generators:
                if self.is_legal(s, move, k_param):
                    next_s = self.step(s, move, k_param)
                    
                    if next_s in target_class:
                        return True
                    
                    if next_s not in visited:
                        visited.add(next_s)
                        frontier.append((next_s, depth + 1))
        
        return False
    
    # ------------------------------------------------------------------------
    # Topology Analysis
    # ------------------------------------------------------------------------
    
    def compute_scc_id(self, state: QAState, generators: List[Tuple[str, int]],
                      max_depth: int = 100) -> str:
        """
        Approximate SCC ID via reachable set hash.
        For exact SCCs, use Tarjan on full graph.
        """
        reachable = self._compute_reachable_set(state, generators, max_depth)
        canonical = min(reachable, key=lambda s: s.primitive)
        return hashlib.md5(str(canonical.primitive).encode()).hexdigest()[:8]
    
    def _compute_reachable_set(self, state: QAState, 
                              generators: List[Tuple[str, int]],
                              max_depth: int) -> Set[QAState]:
        """BFS reachable set computation"""
        visited = {state}
        frontier = [state]
        depth = 0
        
        while frontier and depth < max_depth:
            next_frontier = []
            for s in frontier:
                for move, k_param in generators:
                    if self.is_legal(s, move, k_param):

                        if next_s not in visited:
                            visited.add(next_s)
                            next_frontier.append(next_s)
            frontier = next_frontier
            depth += 1
        
        return visited

    def compute_topology_stats(self, states: List[QAState],
                              generators: List[Tuple[str, int]]) -> Dict:
        """
        Compute topology statistics:
        - #states, #edges, #failures (by type)
        - #SCCs, max SCC size
        
        Args:
            generators: [(move, k_param), ...] 
                       e.g., [("sigma", 2), ("lambda2", 2)]
        """
        num_states = len(states)
        
        # Count edges and failures
        legal_edges = 0
        fail_counts = defaultdict(int)
        
        for s in states:
            for move, k_param in generators:
                if self.oracle.is_legal(s, move, k_param):
                    legal_edges += 1
                else:
                    fail_type = self.oracle.get_fail_type(s, move, k_param)
                    if fail_type:
                        fail_counts[fail_type.value] += 1
        
        # Compute SCCs via Tarjan
        sccs = self._compute_sccs(states, generators)
        num_sccs = len(sccs)
        max_scc_size = max(len(scc) for scc in sccs) if sccs else 0
        
        return {
            'num_states': num_states,
            'num_edges': legal_edges,
            'num_failures': sum(fail_counts.values()),
            'fail_type_counts': dict(fail_counts),
            'num_sccs': num_sccs,
            'max_scc_size': max_scc_size,
            'scc_histogram': self._scc_histogram(sccs)
        }
    
    def _compute_sccs(self, states: List[QAState],
                     generators: List[Tuple[str, int]]) -> List[Set[QAState]]:
        """
        Tarjan's algorithm for exact SCC computation.
        """
        # Build adjacency list
        graph = defaultdict(list)
        state_set = set(states)
        
        for s in states:
            for move, k_param in generators:
                if self.oracle.is_legal(s, move, k_param):
                    next_s = self.oracle.step(s, move, k_param)
                    if next_s in state_set:  # Stay within enumerated set
                        graph[s].append(next_s)
        
        # Tarjan's algorithm
        index_counter = [0]
        stack = []
        lowlinks = {}
        index = {}
        on_stack = defaultdict(bool)
        sccs = []
        
        def strongconnect(node):
            index[node] = index_counter[0]
            lowlinks[node] = index_counter[0]
            index_counter[0] += 1
            on_stack[node] = True
            stack.append(node)
            
            for neighbor in graph[node]:
                if neighbor not in index:
                    strongconnect(neighbor)
                    lowlinks[node] = min(lowlinks[node], lowlinks[neighbor])
                elif on_stack[neighbor]:
                    lowlinks[node] = min(lowlinks[node], index[neighbor])
            
            if lowlinks[node] == index[node]:
                scc = set()
                while True:
                    w = stack.pop()
                    on_stack[w] = False
                    scc.add(w)
                    if w == node:
                        break
                sccs.append(scc)
        
        for node in states:
            if node not in index:
                strongconnect(node)
        
        return sccs
    
    def _scc_histogram(self, sccs: List[Set[QAState]]) -> Dict[int, int]:
        """Return histogram of SCC sizes"""
        hist = defaultdict(int)
        for scc in sccs:
            hist[len(scc)] += 1
        return dict(hist)

Paper 2: Learning Topology in a Discrete QA Control System
    1. Problem Statement 
1.1 Background and Motivation
Paper 1 established that the Quantum Arithmetic (QA) transition system on \mathrm{Caps}(N,N) is a deterministic discrete control system whose reachability, failure modes, and strongly connected component (SCC) structure are exact consequences of the generator algebra. In particular: • Reachability and impossibility are structural properties. • Illegal transitions admit typed failure classifications. • Generator expansion induces sharp connectivity transitions.
However, while these properties are decidable given full knowledge of the transition system, an agent interacting with the system does not have access to this global structure a priori. Instead, the agent must infer topology from partial observations of state–generator interactions.
This motivates the learning problem addressed in this paper.
⸻
1.2 Learning Objective (QAWM)
We define the QA World Model (QAWM) learning problem as follows.
Let: • S \subset \mathbb{Z}^2 denote the QA state space, with primitive coordinates s = (b,e). • \Sigma = {\sigma, \mu, \lambda_2, \nu} denote the generator set. • T : S \times \Sigma \to S \cup {\bot} denote the deterministic transition oracle defined in Paper 1. • \mathcal{F} denote the finite set of failure types.
The agent does not observe T directly. Instead, it may query the environment a finite number of times and observe typed transition outcomes.
⸻
1.3 Observations and Dataset
Each interaction produces a record: (s, g, y_{\text{legal}}, y_{\text{fail}}, s', \Delta I) where: • s \in S is the current state, • g \in \Sigma is the applied generator, • y_{\text{legal}} \in {0,1} indicates legality, • y_{\text{fail}} \in \mathcal{F} \cup {\varnothing} is the failure type if illegal, • s' \in S is the next state if legal, • \Delta I denotes invariant deltas (diagnostic only).
A training dataset D consists of a fixed budget of such records, sampled by random exploration: |D| = M \quad\text{(e.g., } M = 5000\text{)}.
No reward signal, policy gradient, or trajectory supervision is provided.
⸻
1.4 Prediction Tasks
The QA World Model learns the following topological predicates: 1. Legality Prediction \widehat{L}\psi : S \times \Sigma \to {0,1} 2. Failure-Type Classification \widehat{F}\psi : S \times \Sigma \to \mathcal{F} 3. Returnability Prediction For a fixed horizon k and target class \tau \subset S, \widehat{V}_{\psi,k} : S \times \tau \to [0,1] approximating: \mathbb{P}\big(\exists\ \text{return path of length } \le k\big).
Crucially, these predictions concern structure, not trajectories.
⸻
1.5 What Is Being Learned
The unknown quantities are not transition dynamics, but global topological structure, including: • legality boundaries, • failure regions, • SCC membership, • bounded reachability.
Learning converts: \text{Unknown topology} ;;\longrightarrow;; \text{Predictive structural model}.
⸻
1.6 What This Is Not
To avoid confusion with reinforcement learning, we emphasize: • No reward function is defined. • No policy is optimized. • No trajectory likelihoods are modeled. • No value iteration or Bellman backups occur.
Unlike model-based RL, which approximates stochastic dynamics p(s'|s,a), QAWM predicts deterministic topological predicates induced by algebraic constraints.
Failures are not noise; they are theorems.
⸻
1.7 Generalization Criteria
Generalization is defined structurally, not statistically.
A learned QAWM is said to generalize if it: 1. Predicts legality and failure types on unseen states. 2. Correctly predicts return-in-k for states in unseen SCCs. 3. Transfers from \mathrm{Caps}(30,30) to \mathrm{Caps}(50,50) without retraining.

Baselines include: • random prediction, • nearest-neighbor lookup, • no-model heuristics.

        Choose which generator to apply given:
        - Current state s
        - Task tau (target SCC/resonance class)
        - World model qawm (legality/return-in-k predictor)
        """
        # Query QAWM for each generator
        legal_probs = [qawm.predict_legal(s, g) for g in generators]
        return_probs = [qawm.predict_return_in_k(s, g) for g in generators]
        
        # Select generator that maximizes expected progress
        scores = legal_probs * return_probs  # Element-wise
        return generators[argmax(scores)]
The learning loop:
    1. RML policy selects generators using QAWM predictions 
    2. Oracle executes moves, collects success/failure 
    3. Policy updated via REINFORCE/PPO on episode outcomes 
    4. QAWM optionally fine-tuned on new data

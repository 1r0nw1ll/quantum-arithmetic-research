# External Validation Docs

- `GENERALIZATION_LEVEL3_RECOMPUTE.md`: Level-3 recompute from raw MNIST
  data and trained weights (meta-validator test `[29]`).
- `PROMPT_INJECTION_BENCH.md`: Guardrail prompt-injection benchmark
  validation on a licensed external dataset subset with typed obstructions
  and false-negative witnesses (meta-validator test `[30]`).
- `AIID_FAILURE_ALGEBRA_SAMPLE50.md`: Manual F1-F6 stratification of 50 AI
  Incident Database cases, with severity/recurrence analysis and taxonomy
  strain flags.
- `AIID_FAILURE_ALGEBRA_COMPOSITION_V1.md`: Phase-II additive refinement for
  composed failure labels (`primary` + optional `secondary` +
  `composition_form`) with explicit `strain_witness`.

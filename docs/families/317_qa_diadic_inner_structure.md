## The generator formula

For any 3-element set {c₁,c₂,c₃} with L=lcm(c₁,c₂,c₃):

```
n = 2L − (L/c₁ + L/c₂ + L/c₃)

2/n = 1/L + 1/(c₁·n) + 1/(c₂·n) + 1/(c₃·n)
```

## n=101 — the unique case

2/101 = 1/101 + 1/202 + 1/303 + 1/606  
= (1/101) × (1 + 1/2 + 1/3 + 1/6)  
= (1/101) × 2 = 2/101 ✓

The inner sum is exactly 1. The **only** way to express 1 as a sum of 3 distinct unit fractions is **1/2 + 1/3 + 1/6** — there is no other solution. This uniqueness makes n=101 the natural endpoint of the Rhind 2/n table: the largest prime requiring the maximum (4-term) decomposition, anchored to the unique unit-fraction representation of 1.

# QA Hysteresis External Prior Art

## Purpose

This note grounds the QA hysteresis bridge in external scholarly prior art before
any novelty claim is made.

The corrected stance is:

> Hysteresis, hidden order, structured medium response, and memory-bearing field
> dynamics are established research themes. QA's possible contribution is a
> finite arithmetic/orbit/cert representation of that prior structure, tested
> against measured B-H loop data.

This document does not certify a physics claim. It defines the literature
boundary that any future QA hysteresis claim must respect.

## 1. Hidden Order / Latent Order Parameters

Hidden order is not a QA invention. It is standard language in condensed-matter
physics for ordered states whose order parameter is not directly exposed by
ordinary probes.

Mydosh and Oppeneer's review of URu2Si2 is the canonical example: a bulk phase
transition is visible, while the microscopic order parameter remains unresolved.
This matters for QA because it sets the right analogy. QA orbit families should
be framed as candidate finite order-structure coordinates, not as the discovery
that hidden order exists.

Roth and Goodwin also show that local hidden order in disordered crystals can
change electronic and phononic band structures. This is directly relevant to the
Ring35 hysteresis work: apparent disorder or coarse shell response can still
contain structured constraints that affect measured physical behavior.

Relevance to QA:

- QA orbit families may be candidate finite coordinates for latent order.
- QA should not claim that hidden order itself is novel.
- QA claims must be tested as mappings from measured response to finite orbit
  structure.

## 2. Hysteresis as Memory / Operator Theory

Magnetic hysteresis is not a static coordinate problem. It is a history-dependent
operator problem.

Preisach and play models are explicit memory-operator frameworks. They model
magnetic hysteresis as sums or integrals of elementary hysteretic operators with
state thresholds and path dependence. This is the correct conceptual baseline
for QA hysteresis work.

Jiles-Atherton hysteresis is another conventional physical reference class. It
models magnetic hysteresis through magnetization dynamics and material
parameters rather than treating B-H loops as arbitrary curves.

Steinmetz-style empirical scaling is the standard engineering loss baseline. It
does not replace loop-shape physics, but it provides an important reference for
frequency/amplitude loss prediction.

Relevance to QA:

- Static QA coordinates are insufficient unless tied to branch/path memory.
- QA orbit topology must compete against memory-operator baselines, not only
  against generic regression.
- The Ring35 result should be read as a finite-observer residual bridge test,
  not as a discovery that hysteresis has memory.

## 3. Structured Vacuum / Plenum / Zero-Point Field Literature

The idea that "empty" space can have structured electromagnetic response has
mainstream and speculative forms. QA must keep those distinct.

Mainstream or mainstream-adjacent examples:

- QED vacuum response: strong-field QED can treat vacuum response using
  effective permittivity and linear-response/Kramers-Kronig reasoning.
- Zero-point field / stochastic electrodynamics: Cetto and de la Pena argue that
  the random electromagnetic zero-point field is an ontological ingredient in
  quantum phenomena. This is not the same as proving any energy-extraction
  claim, but it is a scholarly precedent for treating vacuum fluctuations as
  physically meaningful.

Speculative boundary:

- "Plenum" or vacuum-energy extraction claims are not validated by the QA
  hysteresis experiments.
- QA must not infer overunity, free-energy extraction, or universal scalar-field
  physics from finite B-H loop residual tests.

Relevance to QA:

- QA can test whether structured residual field/order mappings exist in finite
  measured systems.
- QA should not overclaim energy extraction or physical identity with any
  specific plenum ontology.

## 4. Whittaker / Scalar-Potential Lineage

Whittaker belongs in the lineage, but he must be used precisely.

The 1903 Whittaker paper concerns angular decompositions of wave-equation
solutions and motivates the QA rational direction ladder.

The 1904 Whittaker paper expresses electromagnetic fields using two scalar
potential functions. That is a scalar-potential representation result, not a QA
proof of Maxwell/EM, not a proof of Bearden-style scalar claims, and not a proof
that B-H hysteresis is Whittaker physics.

Relevance to QA:

- The QA Whittaker ladder should be framed as observer-side scalar/angular/path
  representation.
- Whittaker 1903/1904 are source anchors for representation strategy, not
  automatic physical validation.
- QA variables must not be renamed or normalized into standard EM potentials.

## 5. Nonlinear Media / Phase Conjugation / Time Reversal

Nonlinear media and phase-conjugate/time-reversal methods are important
precedents for path-dependent reconstruction.

Phase conjugation in nonlinear optics is often described as a wavefront reversal
or time-reversal-like process. More broadly, wave systems with nonlinear or
history-dependent media can encode path information that is not captured by
static state coordinates.

Relevance to QA:

- Hysteresis residuals should be tested through branch/path-dependent
  reconstruction.
- QA should avoid treating a loop as a static cloud of points.
- The meaningful bridge target is path/order structure in measured response,
  not merely a fitted scalar loss.

## 6. Position of QA

QA does not claim hidden order is new.

QA does not claim hysteresis memory is new.

QA does not claim structured medium response is new.

The QA claim, if supported, is narrower:

> Canonical QA finite-orbit structure may provide a certifiable arithmetic
> representation of known hidden-order and memory-operator phenomena, and may
> capture residual B-H loop structure beyond conventional calibrated shell
> models under specified splits and null tests.

The Ring35 experiments currently support only a limited bridge result:

- Canonical QA orbit-family residual reconstruction slightly improves the main
  split over ordinary H/B shell reconstruction.
- The aggregate 6/12 transfer-win signal is not significant against the tested
  shuffled-label nulls.
- Regime diagnosis suggests the useful signal is conditional, strongest at the
  2000 Hz holdout and selected low/mid amplitude holdouts.

This is a bridge test, not a discovery of hysteresis and not a universal physics
claim.

## 8. Novelty Statement

Draft novelty language:

> The novelty claimed here is not that hysteresis contains hidden order, nor
> that field media can have structured latent response. The novelty is the
> proposed QA finite-orbit representation and cert-gated residual bridge from
> measured B-H loops to canonical QA orbit topology.

Any stronger statement requires a stronger experiment:

- regime-specific null tests;
- comparison to Preisach/play/Jiles-Atherton baselines;
- held-out material/ring transfer;
- explicit failure criteria against shuffled orbit topology;
- documentation of where QA orbit topology wins and where it fails.

beyond conventional H/B shell reconstruction under serious null tests?

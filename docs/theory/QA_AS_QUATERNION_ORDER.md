## 1. The correspondence: QA = M₂(ℤ)

Voight Ch. 3 (Involutions): for `B = M₂(F)` the **adjugate** is the standard
involution `α ↦ ᾱ`; the **reduced trace** `trd = α + ᾱ` is the matrix trace; the
**reduced norm** `nrd = αᾱ` is the determinant; every element satisfies
`α² − trd(α)α + nrd(α) = 0`; and `B¹ = {nrd = 1}` is the norm-1 unit group.

| QA object | Quaternion-algebra object (Voight) |
|---|---|
| the T-operator `M = [[0,1],[1,1]]` | an element of the order **M₂(ℤ)** (`trd=1`, `nrd=−1`) |
| Fibonacci/Lucas orbit structure | `M` is the **golden element**: `M²=M+I`, char poly `x²−x−1` |
| QA Eisenstein norm `f(b,e)=b²+be−e²` | the **reduced norm (det)** restricted to `ℤ[M]` (`det(bI+eM)=b²+be−e²`) |
| `qa_neg` / phase conjugation | the **standard involution (adjugate)** restricted to rotors, `θ↦−θ` |
| state SB-words + `L, R, M²`, certs [294]–[303] | `B¹ = SL(2,ℤ)` = **norm-1 units** (rotors) |

The single load-bearing fact is `M² = M + I`: the QA T-operator satisfies the
golden recurrence, so `ℤ[M] = ℤ[φ]`. QA's Fibonacci behaviour is not an analogy to
the golden ratio — `M` *is* `φ`, realized as an integer matrix.

## 2. The order is maximal; "versor" vs the norm-1 unit group

Auditing the versor certs [294]–[303] against Voight resolves two points (both
machine-verified):

- **The order is M₂(ℤ) itself — the *maximal* order, not a suborder/Eichler
  order.** The ℤ-algebra generated by the QA versors `⟨L,R,M⟩` contains all four
  matrix units (`E₂₁=L−I`, `E₁₂=R−I`, `E₁₁=E₁₂E₂₁`, `E₂₂=E₂₁E₁₂`), so it *is*
  M₂(ℤ), the unique maximal order of the split algebra M₂(ℚ). No Eichler-order
  subtlety arises at this level.
- **"Versor" in the certs is the Hestenes/Clifford (grade-tracked) sense, broader
  than Voight's `B¹`.** The norm-1 unit group `B¹ = SL(2,ℤ)` is the **even-grade /
  rotor** part. `M` has `nrd = −1`, so `M ∉ B¹` — it is an *odd-grade* versor (a
  reflection); `B¹` is generated by `L, R` and `M²`. The certs track this
  correctly ([294]: "M has det=−1 (NOT SL(2,ℤ)) but M²=L·R (IS SL(2,ℤ))"). Rule of
  thumb: **#M-factors even ⇔ nrd = +1 ⇔ in B¹ (rotor).**

## 3. The unification of two cert clusters

The M₂(ℤ) identification shows two separately-developed cert clusters to be one
object seen two ways:

- **Versor / SL(2,ℤ) certs [294]–[303]** live in **M₂(ℤ)** — the split quaternion
  order. Versors are its norm-1 units; two T-steps `M² = LR` are one SL(2,ℤ) rotor.
- **Orbit / Fibonacci-Frobenius certs [384]–[431]** live in **ℤ[M] = ℤ[φ] =
  O_{ℚ(√5)}** — the *commutative subring* generated by the T-operator (since
  `M² = M + I`). Their `O_{ℚ(√5)}` number theory *is* the arithmetic of this
  subring.
- The **QA norm form** `f(b,e)=b²+be−e²` is the **reduced norm restricted to
  ℤ[M]** — simultaneously the golden-order norm and the quaternion reduced norm on
  that commutative slice. It is the bridge between the clusters.

So `ℤ[φ] ⊂ M₂(ℤ)`: the golden quadratic order sits inside the split quaternion
order as the subring generated by the T-operator.

**Phase conjugation** (certs [518]–[521], the channel equalizer, the sympathetic-
resonance demo) is the **standard involution restricted to the rotor subgroup**,
`θ↦−θ` — the abelian shadow of the full non-commutative adjugate on M₂. The
four-wave-mixing phase-sum `θ_c = θ_f + θ_b − θ_s` is rotor composition with
conjugation. The whole cluster sits inside Voight's framework as this restriction.

## 4. Integrality: why fractional `(b,e)` leaves the order

The framework is exact by axiom **S2** (state is `int`/`Fraction`, never float).
Quaternion-theoretically the answer splits by level:

- **Algebra level — the datatype is irrelevant.** The identities that make QA
  quaternionic (standard involution = adjugate, `trd`=trace, `nrd`=det, char poly,
  `M²=M+I`) hold over *any* field, hence for `Fraction (b,e)` exactly as for `int`
  (verified for `(1/3,2/5)`, `(−7/4,9/8)`).
- **Arithmetic level — integrality is everything.** The structure the grounding is
  *about* — the order M₂(ℤ)/ℤ[φ], versors = SL(2,ℤ), the Fibonacci orbit lattice,
  class number, Brandt matrices — is an **integrality** phenomenon. In QA's `(b,e)`
  basis the element is `b + eφ`, and the maximal order `O = ℤ[φ]` is **exactly the
  integer `(b,e)` pairs**. A genuinely non-integer `(b,e)` (e.g. `(1/2,0)`, with
  `nrd = 1/4 ∉ ℤ`) leaves the order for the ambient *field* ℚ(√5) / algebra M₂(ℚ),
  where there is no lattice, no discrete unit group, no class number, no Brandt
  structure.

So: **it is integrality, not the Python type, that matters.** `Fraction(3)` is an
order element; `Fraction(1,2)` is a field element outside the order. Uniform
scaling `(1/n)·O` is a fractional ideal *homothetic* to `O` (same ideal class), but
a single mixed-denominator `(b,e)` lies in no homothety of `O` and carries no order
structure. This is the number-theoretic content of S2: for the arithmetic layer,
the exact value must also be an algebraic integer.

| **3** | **inert** in ℤ[φ] (not ramified) | `x²−x−1` irreducible mod 3 → `ℤ[φ]/3 = 𝔽₉`; `M mod 3` has order `π(3)=8` | the 3-adic *filtration* [301] and the `3∣m` collapse [515] |

## 6. Brandt/Hecke theory: the obstruction and its resolution

### 6.1 Why M₂(ℤ)'s own Brandt theory is degenerate

Voight opens Ch. 41: *"When B is indefinite … the class set is trivial. We are
left with the case that B is definite."* Brandt matrices
`T(n)ᵢⱼ = #{J ⊂ Iⱼ : nrd(J)=n·nrd(Iⱼ), [J]=[Iᵢ]}` — the adjacency matrices of the
`n`-Brandt graph on the ideal-class set, and the Hecke operators on the resulting
modular forms — carry content **only when `B` is definite**.

QA = **M₂(ℚ) is split/indefinite** (its golden norm form `b²+be−e²` has signature
`(1,1)`). By strong approximation the maximal order M₂(ℤ) then has class number
`h = 1`, so every Brandt matrix is `1×1`: `T(p) = (p+1)`, the trivial Eisenstein
eigenvalue, no cusp forms. *This is a structural fact about the split algebra, not
a limitation of QA.* What survives, tied to existing certs:

- The **`p`-neighbor combinatorics** are real: exactly `p+1` index-`p` sublattices
  of ℤ² (the `(p+1)`-regular Bruhat–Tits tree that `Tₚ` sums over). But the QA
  T-operator `M` is a *single* GL₂(ℤ) element, not the Hecke sum — orbit dynamics
  `⟨M⟩` is *orthogonal* to the neighbor/Hecke structure.
- The **3-adic filtration [301]** is the *central/scalar* line of the `p=3` tree
  (`v↦3v`, index `9=3²`, the `T(p²)` diagonal term), iterated — one line, not the
  branching.
- The **QA resonance matrix** `einsum('ik,jk->ij', T, T)` is a single symmetric
  **Gram** matrix, *not* a Brandt matrix (not a commuting family of neighbor
  *counts* with column sums `p+1`). The analogy is superficial; recorded so it is
  not reasserted.

### 6.2 The resolution: the definite golden order = the icosian ring = E8

The interesting golden Hecke eigenvalues QA computes — the CM Hilbert modular form
**LMFDB 2.2.5.1-125.1-a over ℚ(√5)**, cert cluster [384]–[431] — are exactly what a
Brandt construction should reproduce. By Jacquet–Langlands that requires a
**definite** quaternion algebra over ℚ(√5), not the split M₂/ℚ.

The canonical one is `B = (−1,−1 | ℚ(√5))` (`i²=j²=k²=−1`, `ij=k`, reduced norm
`w²+x²+y²+z²`), **totally definite** (the norm is a sum of four squares →
totally positive). Its maximal order is the **icosian ring** (Conway–Sloane,
*SPLAG* Ch. 8), verified exactly (`qa_icosian_order.py`, 7/7):

- its **120 units** all have reduced norm 1, are algebraic integers in ℤ[φ], and
  are closed under multiplication — the **binary icosahedral group 2I**;
- under one real embedding those 120 units are the **600-cell** (each vertex has
  exactly 12 nearest neighbors at inner product `φ/2`; inner-product spectrum
  `{±1, ±φ/2, ±½, ±1/2φ, 0}`, the golden-angle geometry);
- the icosian ring is isometric to **E8** (*SPLAG*): the 600-cell's 120 vertices
  plus a φ-scaled copy give E8's 240 roots. (Verified in-session: total
  definiteness, `120 units = 2I`, integrality, the 600-cell. The full E8 isometry
  is cited; the plain trace form on the ℤ-basis is a `det 5⁴` rescaling of E8.)

**Why this matters for QA:** ℚ(√5) *is* QA's own field (`ℤ[M]=ℤ[φ]=O_{ℚ(√5)}`), so
the definite order carrying QA's golden Hecke eigenforms is built on the golden
ratio itself — and it is the **same E8** QA already aligns to (4D tuples → 8D →
240 roots). The QA↔E8 alignment stops being an imposed projection and *becomes* the
icosian/E8 structure of the definite golden order.

### 6.5 The tower: QA reaches ℚ(√5); the Hecke eigenvalues live at ℚ(ζ₅)

§6.1 said QA's orbit generator `M` is *orthogonal* to the Hecke structure. This
sharpens that into a precise statement about the cyclotomic tower
`ℚ ⊂ ℚ(√5) ⊂ ℚ(ζ₅)` (`qa_fibonacci_hecke_compare.py`), answering the question the
whole arc was really after: does QA's own Fibonacci–Frobenius dynamics *compute* the
CM form's Hecke eigenvalues, or only share the field?

QA's Fibonacci–Frobenius system (cert [423]) is the arithmetic of `φ`, the roots of
`x²−x−1` mod `p` — the **ℚ(√5)** (degree-2) layer. Its Frobenius **trace `φ+ψ ≡ 1`
for every split prime** (Vieta): a constant, the fixed golden recurrence, carrying
**no varying GL₂ eigenvalue** — only the splitting and the GL₁ order datum
`α(p)=ord(φ/ψ)`. The CM form's Hecke eigenvalue `a_𝔭 ∈ ℤ[φ]` (from the from-scratch
Brandt cusp factors: `𝔭₁₁ → 5φ−3`, `𝔭₃₁ → 5φ−8`) is instead a **ℚ(ζ₅)** (degree-4)
CM-character value, **nonzero iff `p ≡ 1 mod 5`** — splitting in `ℚ(ζ₅)/ℚ(√5)`, a
*finer* condition than splitting in `ℚ(√5)` (`p ≡ ±1 mod 5`).

The decisive datum: `p=11` (`≡1`) and `p=19` (`≡4`) **both split in ℚ(√5)** with the
same Fibonacci "split" type (trace 1), yet `a₁₁ = 5φ−3 ≠ 0` while **`a₁₉ = 0`**. QA's
`φ`-arithmetic cannot distinguish them; the CM form does. Therefore:

> **QA's Fibonacci–Frobenius dynamics computes the eigenvalue *field* ℚ(√5) and the
> splitting — not the Hecke eigenvalues, which are ℚ(ζ₅) CM data one cyclotomic layer
> up.** The Brandt/Hecke bridge is field-level + Jacquet–Langlands (through the
> definite icosian/E8 order), *not* QA-orbit-dynamical. That is the honest content of
> "the golden order carries QA's field, the definite order carries the forms."

This is a *boundary*, not a defeat: it says exactly where QA sits in the tower (the
real quadratic floor) and why the modular eigenvalues require ascending to the
cyclotomic layer the definite order supplies.

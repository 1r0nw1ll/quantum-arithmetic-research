
# QA Harmonic FFT – Symbolic Spectral Engine

## Overview
This symbolic FFT alternative uses Quantum Arithmetic tuples (b, e, d, a) and rational trigonometry to produce exact, integer-preserving spectral outputs.

## Files Included
- `qa_fft.py`: The symbolic transform core
- `qa_tuples.csv`: Canonical QA input sequence
- `qa_fft_output.csv`: Transform results

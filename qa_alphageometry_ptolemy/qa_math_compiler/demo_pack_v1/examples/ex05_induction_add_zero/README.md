# ex05_induction_add_zero

This example is a compact induction narrative over Nat equality.
Open `claim.txt` for the informal claim and Lean target statement.
`task.json` sets a proof budget that allows explicit induction tactics.
`trace.json` records a successful induction-style compilation trace.
`replay.json` confirms deterministic replay for that one trace.
`pair.json` binds the human claim to the exact formal goal string.

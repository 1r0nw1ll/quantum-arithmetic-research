1. THE QA NORM ON DISCRETE GENE COUNTS

Standard analysis treats gene expression as continuous magnitudes.
But the underlying events are discrete: which genes flip, how many,
in what directional pattern. QA operates on this discrete layer.

We applied the QA quadratic norm f(b, e) = b^2 + be - e^2 to the
discrete gene counts from Supplementary Data 2, where b = number
of upregulated genes and e = number of downregulated:

  extract 0hr:  (2, 8)  ->  f = -44   v_3 = 0   COSMOS
  extract 4hr:  (11, 20) -> f = -59   v_3 = 0   COSMOS
  atp 0hr:      (2, 2)  ->  f = 4     v_3 = 0   COSMOS
  atp 4hr:      (3, 0)  ->  f = 9     v_3 = 2   SATELLITE

The 3-adic valuation v_3(f) classifies orbits exactly in mod-9 QA:
v_3 = 0 is Cosmos, v_3 = 2 is Satellite, v_3 >= 4 is Singularity.
This is not a statistical heuristic — it is the ideal factorization
of the norm in Z[phi]/9Z[phi], classifying all 81 state pairs with
zero error.

The ATP 4-hour result: three genes upregulated (Nr4a1.L, Nr4a1.S,
LOC121395420), zero downregulated. f(3, 0) = 9 = 3^2. v_3 = 2.
Satellite classification from the discrete gene count alone.

4. THE CRITICAL PRIME: WHY 3 APPEARS EVERYWHERE

The number 3 is structurally privileged in mod-9 QA because 9 = 3^2.
The Satellite orbit consists entirely of pairs with coordinates that
are multiples of 3: {(3,3), (3,6), (3,9), (6,3), (6,6), (6,9),
(9,3), (9,6)}.

This generates a testable rule: for any stimulus producing a
unidirectional gene response (n genes up, 0 down), the QA norm is
f(n, 0) = n^2, and v_3(n^2) = 2 * v_3(n). So the response classifies
as Satellite whenever n is a multiple of 3 (3, 6, 9, 12, ...) and
as Cosmos otherwise (1, 2, 4, 5, 7, 8, 10, 11, ...).

More generally, f(n_up, n_down) mod 3 = 0 marks the Satellite
boundary. The Satellite-compatible small pairs are:
  (3,0), (0,3), (3,3), (6,0), (0,6), (3,6), (6,3), (6,6), (9,0)...

This is testable across your entire stimulus library: count
